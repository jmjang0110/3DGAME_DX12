

단일 메쉬 
	육면체와 같이 하나의 메쉬를 가지고 있는 게임 오브젝트이다. 


다중 메쉬 
	- 메쉬들을 공간적인 계층 구조 ( Spatial Hierarchy ) 또는 프레임 계층 구조

	- Spatial Hierarchy 
		부모-자식 관계 
		트리구조
		노드(Node, Frame) - 노드는 게임 오브젝트이다. ( 행렬 + 메쉬 ) + 자식에 대한 포인터들 
		 -> 부모 좌표계의 영향을 받는다. ( 상대적이다 )
	- Frame of Reference 
		물리학 수학에서 방향을 가진 점을 표현하기 위하요 사용 ( 좌표계 )


	모든 트리는 이진트리로 바꿀 수 있다 .
